# 共有フォルダ セットアップ

共有フォルダのグループ作成・権限設定手順をまとめたドキュメントです。

---

## 1. グループの作成

「コンピューターの管理」からローカルグループを作成します。

### 開き方（いずれかの方法）

- **方法A**: Windowsマーク > 右クリック > コンピューターの管理 > 「ローカルユーザーとグループ」 > グループ
- **方法B**: `Win + R` で以下を実行

```
lusrmgr.msc
```

「グループ」フォルダ内で右クリック → 「新しいグループ」から空のグループを作成できます。

---

## 2. グループへのメンバー一括追加（PowerShell 管理者実行）

以下のスクリプトをコピペして実行してください。  
`$groupName` と `$userListPath` は環境に合わせて書き換えてください。

```powershell
$groupName = "RD_ALL"

# ユーザーリストのテキストファイルの場所を設定
$userListPath = "G:\マイドライブ\3DPDF_PK7376_フォルダ権限\フォルダ権限リスト - RD_ALL.txt"

# テキストファイルからユーザーリストを読み込んで、1人ずつグループに追加する処理
Get-Content $userListPath | ForEach-Object {
    try {
        Add-LocalGroupMember -Group $groupName -Member $_
        Write-Host "ユーザー '$_' をグループ '$groupName' に追加しました。"
    } catch {
        Write-Warning "ユーザー '$_' を追加できませんでした。ユーザーが存在しないか、既に追加済みです。"
    }
}
```

---

## 3. 共有フォルダの権限設定

スクリプト実行後、以下の手順で共有フォルダにグループを設定します。

1. 対象フォルダを右クリック > **プロパティ** > **共有** タブ
2. 共有者にグループ（例: `RD_ALL`）を追加する
3. 「探す場所」のドメインを**本番環境のサーバー名**に変更する

---

## 4. 別ドメインのユーザーを追加する場合

ドメインが異なるユーザー（例: タイメンバー等）は `RD_ALL` の設定に引っかからないため、**個別に設定**する必要があります。

1. フォルダ右クリック > **プロパティ** > **共有**
2. 検索バーで「ユーザーの検索」を開く
3. ドメインを対象のドメイン（例: `lb`）に変更する
4. ユーザーIDを入力し、**「名前の確認」を押してから**実行する

> **注意**: 「名前の確認」を押さずに実行すると正しく反映されません。

複数ユーザーを一度に設定する場合は、セミコロンで区切ります:

```
(ユーザーID);(ユーザーID)
```
