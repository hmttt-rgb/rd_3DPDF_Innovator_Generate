1. グループを作成
 「コンピューターの管理」でグループを作成する

追記：2026/01/06
「コンピュータの管理」：Windowsマーク> 右クリック> コンピュータの管理> 
                        「ローカルユーザとグループ」> グループ

もしくは、
Windowsマーク+R > "lusrmgr.msc"と検索> グループ

いずれについても、適当なところ右クリックすることで、空のグループを作成できる
 


2. PowerShell(管理者)で実行
  以下をコピペし、フォルダ>プロパティ>共有 から共有者を設定する
  探す場所のドメインを本番環境のサーバー名に変更する


追記：2026/01/13
3. ドメインが違うと(タイメンバーなど)RD_ALLの設定に引っかからない→個別に設定することで解決した

フォルダ右クリック> プロパティ> 共有> 検索バー"v"「ユーザーの検索」> ドメインをlbに変える> 
以下を空白にコピペする(追加メンバーがいれば付け足す)
> 「名前の確認」を押してから実行しないとうまく反映されない

複数設定の場合はユーザーIDの間にセミコロンを入れる。
(ユーザーID);(ユーザーID)


# ----------------------------------------------------------

$groupName = "RD_ALL"

# ユーザーリストのテキストファイルの場所を設定
$userListPath = "G:\マイドライブ\3DPDF_PK7376_フォルダ権限\フォルダ権限リスト - RD_ALL.txt"

# テキストファイルからユーザーリストを読み込んで、1人ずつグループに追加する処理
Get-Content $userListPath | ForEach-Object {
    try {
        Add-LocalGroupMember -Group $groupName -Member $_
        Write-Host "ユーザー '$_' をグループ '$groupName' に追加しました。"
    } catch {
        Write-Warning "ユーザー '$_' を追加できませんでした。ユーザーが存在しないか、既に追加済みです。"
    }
}
