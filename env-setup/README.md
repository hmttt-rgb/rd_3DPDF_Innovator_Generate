# Environment Setup

This directory contains all configuration files and scripts needed to rebuild the server and client environments from scratch.

## Contents

| Path | Description |
|------|-------------|
| `gpu-dda/` | Windows GPU (Discrete Device Assignment) settings for the Virtual Machine |
| `app-configs/` | Detailed Application Settting (Acrobat, Bullzip, NVIDIA etc...)|
| `shared-folders/` | Shared folder mount points and permission definitions |
| `task-scheduler/` | Windows Task Scheduler XML export files |
